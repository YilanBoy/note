# Ansible

Ansible 是一套知名的 IaC (Infrastructure as Code) 工具

有別於用來建立資源的 Terraform，Ansible 主要用在設定機器上的環境

Ansible 最大的特點在於它是 Agentless 的，你不需要在遠端主機上安裝任何 Agent 就能操作遠端主機
